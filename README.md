# 🚀 Plataforma de Estudos Dev

Bem-vindos ao projeto oficial da nossa Software House! Este ecossistema está sendo desenvolvido para organizar trilhas de estudo, compartilhar recursos e fortalecer o portfólio de todos os membros.

## 🎯 Objetivo
Transformar a teoria da faculdade em prática real, simulando o ambiente de uma Software House profissional.

## 🛠️ Tecnologias Previstas
- **Frontend:** React / Next.js
- **Backend:** Node.js / TypeScript
- **Banco de Dados:** Supabase / PostgreSQL
